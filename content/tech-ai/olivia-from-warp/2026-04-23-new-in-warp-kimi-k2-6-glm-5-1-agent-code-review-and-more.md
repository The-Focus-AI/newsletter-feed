New Models: Kimi K2.6 and GLM 5.1
---------------------------------

Warp Agent now supports Kimi K2.6 and GLM 5.1, giving you two leading open-weight models to try for coding and agentic workflows.

New Models: Kimi 2.6 and GLM 5.1

-----------------------------
Remote Control for CLI agents
-----------------------------

Publish a running third-party agent session — such as Claude Code, Codex, or OpenCode — to the cloud with a single click. Once published, you can monitor progress, review output, and steer the agent from your phone, a web browser, or another computer. Learn more ( https://docs.warp.dev/agent-platform/third-party-agents/remote-control )

Run Agents Remotely

--------------------------
Code Review for CLI Agents
--------------------------

Leave inline code review comments on changes made by Claude Code, Codex, and other CLI agents. Just like reviewing a PR. Learn more ( https://docs.warp.dev/agent-platform/third-party-agents/overview )
