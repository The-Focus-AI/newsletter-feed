Google adds Agentic Vision to Gemini 3 Flash, enabling multi-step, code-driven reasoning on images
5,274 Likes
Traditional image understanding works in one pass. If the model misses a small object or dense text, the answer degrades. Agentic Vision changes the flow by turning vision into an active investigation that uses code as part of reasoning.
The core idea is simple. The model does not just look at an image. It plans steps, executes Python code to modify or analyze the image, then looks again. This loop grounds answers in pixels instead of latent guesses.
Google reports a
5-10% quality increase across most vision benchmarks
when you enable this mode.
How it works
Uses a Think-Act-Observe loop to plan actions and reprocess visual evidence
Executes Python code to crop, zoom, annotate, count objects, or compute values
Appends modified images to the context window before producing the final answer
What you can do
Inspect fine-grained regions in high-resolution images
Draw bounding boxes and labels to avoid counting errors
Run deterministic visual math and generate plots with Matplotlib

DeepSeek ships OCR 2, an open-source VLM using semantic token ordering for documents
3,017 Likes
DeepSeek releases DeepSeek-OCR 2 to fix a long-standing weakness in OCR systems: reading complex documents in the wrong order.
Traditional vision models scan images like pixels on a screen, left to right, top to bottom. DeepSeek-OCR 2 changes that behavior by teaching the model to read documents more like a human.
At the core sits DeepEncoder V2. Instead of locking the model into a grid, the encoder first builds a global understanding of the page, then decides what to read first, next, and later. This shift matters for real documents where columns, tables, and forms break simple layouts.
What it does
Performs OCR, document understanding, and visual reasoning in a single model
Reorders visual tokens using semantic reading order rather than fixed raster scans
Why it matters
Avoids column hopping in multi-column documents
Preserves row-column structure when reading tables
Maintains label-value relationships in forms
