Now, let's cover each individual launch in detail.

DAY 1: AUTOMATIONS

https://resend.com/blog/introducing-automations

On Monday, we launched Automations, a new way to orchestrate lifecycle emails triggered by events in your app.

With Automations, you can:

 * Define a trigger from an event in your app

 * Personalize each step with delays, waits, and conditions

 * Get full visibility into every run in real time

→ Read the announcement https://resend.com/blog/introducing-automations

DAY 2: AI EMAIL EDITOR

https://resend.com/blog/ai-email-editor

On Tuesday, we announced the new Resend email editor, extended by AI.

The editor helps you:

 * Start with your existing content by turning a URL into an on-brand draft

 * Bring your own agentic stack via the Resend MCP server

 * Catch mistakes before you hit send

→ Read the announcement https://resend.com/blog/ai-email-editor

DAY 3: RESEND CLI 2.0

https://resend.com/blog/resend-cli-2

On Wednesday, we shipped version 2.0 of the Resend CLI with powerful new features:

 * Send emails using local React files

 * Manage Automations and events from your terminal

 * Skills for AI agents to interact with the Resend API

The CLI now gives you access to 50+ commands across the full Resend API, without leaving your terminal.

→ Read the announcement https://resend.com/blog/resend-cli-2

DAY 4: CUSTOM TRACKING DOMAINS

https://resend.com/blog/introducing-custom-tracking-domain

On Thursday, we launched custom tracking domains, free for all Resend users.

Tracking through a domain you own:

 * Keeps your sender reputation isolated from shared infrastructure

 * Aligns with your sending domain for better deliverability

 * Shows your recipients a familiar domain on every link

It works for both click tracking and open tracking, and can be configured through the Dashboard or the API.

→ Read the announcement https://resend.com/blog/introducing-custom-tracking-domain

DAY 5: REACT EMAIL 6.0

https://resend.com/blog/react-email-6

On Friday, we announced a new major version of react.email https://react.email/, featuring a new open-source email editor you can embed directly in your own app.

React Email 6 includes:

 * An embeddable editor that you can drop into your app with one line of code

 * New templates for authentication and e-commerce flows

 * Unified package for easier integration

→ Read the announcement https://resend.com/blog/react-email-6

DAY 6: RESEND FORWARD

https://resend.com/blog/resend-forward

On Saturday, we announced Resend Forward https://resend.com/forward, our first-ever conference. By builders, for builders.

Join us in San Francisco on October 21st for a full day focused on the craft behind building software. We'll hear from leaders working at the edge of software and AI, with a special emphasis on practical tools and ideas you can actually use.

→ Join the waitlist https://resend.com/forward
