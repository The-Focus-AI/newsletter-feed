# [Product Update] Name update for Vertex AI and Data and Analytics products starting Apr 22, 2026

**From:** Google Cloud <CloudPlatform-noreply@google.com>
**Date:** Wed, 22 Apr 2026 19:33:00 -0700

---

Your projects, configurations, and pricing will remain unchanged.

MY CONSOLE

Hello Will,

On April 22, 2026, at Google Cloud Next, we announced the evolution of the  
Vertex AI Platform into the Gemini Enterprise suite. As part of this  
transition, Vertex AI Platform is now part of the Gemini Enterprise Agent  
Platform.

Additionally, we have updated several Data Analytics product names to be  
more descriptive and easier to understand. These changes include:

Dataplex Universal Catalog is now Knowledge Catalog
BigLake is now Lakehouse
Dataproc is now Managed Service for Apache Spark
Composer is now Managed Service for Apache Airflow
Looker Studio is now Data Studio

What you need to know

Your projects, configurations, and pricing will remain unchanged. The name  
changes will be reflected across your projects, Google Cloud console,  
documentation, legal terms, and pricing pages over the coming months. See  
below for more details.

Here are the three key updates to expect:

Product and feature name changes as outlined in the release notes: These  
name changes will appear across Google Cloud console, Gemini Enterprise  
Agent Platform documentation, Data Analytics documentation, legal terms,  
and pricing pages.
Google Cloud console navigation changes: The navigation menus under Agent  
Platform (formerly Vertex AI) and Data Analytics have been updated to  
centralize agentic products and features. Bookmarked links will continue to  
work via automatic redirects. You can find more information about this  
change in the Gemini Enterprise Agent Platform release notes.
SKU display name and description update: SKU display names and descriptions  
will be updated in the coming months. Updated SKUs are listed in the Google  
Cloud Platform SKUs page. Data Analytics SKU names will remain unchanged.  
In all cases, your billing or invoice will remain unchanged.

This name change will reflect on your affected projects in the attachment.

What you need to do

No action is required from you. The updates listed above will reflect on  
your projects and billing effective April 22, 2026.
