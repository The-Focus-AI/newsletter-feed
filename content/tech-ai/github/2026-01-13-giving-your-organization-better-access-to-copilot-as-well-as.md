Making Copilot more efficient to use
------------------------------------

* Use Agent Skills to teach Copilot new tricks: ( https://github.blog/changelog/2025-12-18-github-copilot-now-supports-agent-skills/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Create folders of instructions, scripts, and resources that Copilot can use to enhance its performance with specialized tasks.
* Hand off tasks to background and cloud agents in VS Code: ( https://code.visualstudio.com/updates/v1_107?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0#_continue-tasks-in-background-or-cloud-agents ) Seamlessly transition tasks between local and cloud agents.
* Let Copilot pick the model for you: ( https://github.blog/changelog/2025-12-10-auto-model-selection-is-generally-available-in-github-copilot-in-visual-studio-code/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Use auto model selection to have Copilot select a model for you based on current model availability.
* *Better experience assigning tasks to GitHub Copilot:* Keep better track of issues when you assign tasks to Copilot since it automatically adds you as an assignee ( https://github.blog/changelog/2025-12-18-assigning-github-copilot-to-an-issue-now-adds-you-as-an-assignee/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ). Plus, you can use the API to assign issues directly to Copilot ( https://github.blog/changelog/2025-12-03-assign-issues-to-copilot-using-the-api/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ).
* Configure your organization so anyone can use Copilot code review: ( https://github.blog/changelog/2025-12-17-copilot-code-review-now-available-for-organization-members-without-a-license/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Enable anyone in your organization to use Copilot code review on pull requests.

( https://github.blog/changelog/2025-12-10-the-github-mcp-server-adds-support-for-tool-specific-configuration-and-more/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 )

* Configure your GitHub MCP Server specifically for tools: ( https://github.blog/changelog/2025-12-10-the-github-mcp-server-adds-support-for-tool-specific-configuration-and-more/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Use the new configuration options for the GitHub MCP Server to load exactly the tools you need with no additional bloat.

Adding support for more models
------------------------------

* Try out Claude Opus 4.5: ( https://github.blog/changelog/2025-12-03-claude-opus-4-5-is-now-available-in-visual-studio-jetbrains-ides-xcode-and-eclipse/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) The latest Claude Opus model is now available in public preview in Visual Studio, JetBrains IDEs, Xcode, and Eclipse.
* Gemini 3 Pro is now available: ( https://github.blog/changelog/2025-12-12-gemini-3-pro-is-now-available-in-visual-studio-jetbrains-ides-xcode-and-eclipse/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) This model is now in public preview for users of Visual Studio, JetBrains IDEs, Xcode, and Eclipse.
* *Use some new models from OpenAI including GPT-5.1-Codex-Max ( https://github.blog/changelog/2025-12-04-openais-gpt-5-1-codex-max-is-now-in-public-preview-for-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) and GPT-5.2 ( https://github.blog/changelog/2025-12-11-openais-gpt-5-2-is-now-in-public-preview-for-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=FY26JAN-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) :* These new models from OpenAI are now available in public preview to use across the Copilot landscape.
