Claude now connects to Blender, Adobe, Canva, Ableton, and more creative tools
7,294 Likes
Claude just got
9 new connectors
for creative tools, and this one is worth paying attention to.
The idea is simple: instead of switching between apps, you stay in Claude and talk to your tools directly. The Blender connector gives you a natural-language interface to its Python API, meaning you describe what you want instead of writing scripts from scratch. You can analyze and debug entire Blender scenes, build custom scripts to batch-apply changes to objects, and add new tools directly to Blender's interface.
Here's the full list of what launched:
Adobe Creative Cloud
: orchestrates multi-step workflows across Photoshop, Illustrator, Firefly, Premiere, and more
Autodesk Fusion
: create and modify 3D models through conversation
Affinity by Canva
: automates batch image adjustments, layer renaming, and file export
Splice
: search its catalog of royalty-free samples from within Claude
Ableton, SketchUp, and Resolume also included
Because the connector is built on MCP, it is accessible to other AI models too, not just Claude. To get started, install any connector from Claude's Settings under Connectors.

OpenAI ships open-source voice app built on gpt-realtime-1.5
5,755 Likes
OpenAI just dropped
realtime-voice-component
, an open-source React package that lets users control your app with their voice in real time.
Here is the core idea. Instead of clicking buttons, users just talk. The app listens, understands the intent, and updates its own state. Your app stays in charge of what actually changes.
What you can build with it:
Forms users fill out by speaking
Theme or settings switchers triggered by voice
Multi-step flows where voice replaces tapping through screens
It runs on
gpt-realtime-1.5
under the hood. You define narrow actions your app can take, the voice layer calls them, and your existing state logic handles the rest. No rewiring your whole app.
To try it: fork the repo, add your
OPENAI_API_KEY
, run
npm run demo
, and you will see a live chess and form demo instantly. The repo is Apache-2.0 licensed, so it is free to use and modify.

Google Gemma 4 powers a fully local browser agent that needs no server
5,422 Likes
An AI agent that lives
entirely inside your browser
. No cloud. No API keys. No server bills. Just your GPU doing the work, locally.
It runs on
Gemma 4 E2B
, a small but capable AI model converted to run natively in Chrome using WebGPU (your graphics card doing AI math). Transformers.js handles the heavy lifting so the model never leaves your machine.
Here is what you can actually do with it:
Ask it to find something from your browsing history using plain English, no exact keywords needed
Ask it to summarize or search the page you are currently on
Tell it to open, switch, or close tabs by just describing what you want
Models download once on first use, then stay cached. The
privacy angle
is real: your tabs, history, and page content never leave your device.
To try it: install from the Chrome Web Store or clone the repo, run
pnpm install
and pnpm run build, then load the dist folder as an unpacked Chrome extension.
